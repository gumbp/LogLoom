# LogLoom: Type-Aware Residual Entropy Coding for Large-Scale Log Compression
***Abstract:*** The rapid growth of logs in modern software systems has made log compression essential for reducing long-term storage costs. Existing work mainly extracts templates to remove explicit structural redundancy. As template redundancy becomes increasingly exploited, however, the compression bottleneck has shifted to log variables. These fields contain dynamic data with heterogeneous types and time-varying drift. Existing coding strategies therefore struggle to model their complex conditional distributions, leaving residual redundancy and causing mismatch under traffic changes and system evolution. Here we propose LogLoom, a lossless log-compression framework for heterogeneous variables and distribution drift. LogLoom first decouples variable types to simplify probabilistic approximation across fields. It then combines type-aware reversible residual transformation with lightweight conditional probability coding. This design maps high-entropy variables into lower-entropy representations without information loss. For online non-stationary settings, LogLoom further uses drift-aware local adaptation to reduce model mismatch and stabilise compression performance.Experiments on the LogBase dataset show that LogLoom achieves an average compression ratio 2.8 times that of the best existing baseline. It also provides greater robustness to temporal drift, indicating that type-aware entropy reduction and local dynamic adaptation can help overcome current bottlenecks in lossless log compression.
